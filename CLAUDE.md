# 딸기마켓 프로젝트

## 프로젝트 소개
당근마켓과 비슷한 중고거래 웹사이트 **딸기마켓**의 개발 연습용 프로젝트입니다.

## 기술 스택
- **Next.js** — 웹 페이지를 만드는 프레임워크 (React 기반)
- **Supabase** — 로그인, 데이터베이스 등 서버 기능을 쉽게 쓸 수 있는 서비스
- **Tailwind CSS** — 디자인을 빠르게 입히는 도구
- **Vercel** — 만든 웹사이트를 인터넷에 올려서 누구나 볼 수 있게 해주는 배포 서비스

## Claude에게 당부하는 것들

### 언어 & 설명 방식
- 개발 용어는 반드시 쉬운 말로 풀어서 설명할 것
  - 예) "라우팅" → "페이지 주소 연결", "컴포넌트" → "재사용 가능한 화면 조각", "마이그레이션" → "데이터베이스 구조 변경"
- 지금 무엇을 하고 있는지 자주 짧게 말해줄 것
  - 예) "로그인 페이지 버튼 색상을 바꾸고 있어요", "데이터베이스에 상품 테이블을 추가하고 있어요"

### 작업 방식
- 사용자가 지시한 것만 단계적으로 수행할 것 — 요청하지 않은 기능을 임의로 추가하지 않음
- 코드로 할 수 있는 것은 최대한 직접 처리할 것
- 꼭 사람이 직접 해야 하는 작업(예: Supabase 대시보드 클릭, Vercel 설정 등)은 쉬운 말로 단계별 안내할 것
  - 예) "지금 Supabase 사이트에서 버튼을 클릭해야 해요. 순서대로 따라해 주세요: 1. ... 2. ..."

### 작업 완료 후
- 개발이 끝나면 항상 브라우저에서 바로 확인할 수 있도록 해당 페이지를 열어줄 것
- 화면을 캡처해서 결과가 잘 나왔는지 직접 확인할 것
